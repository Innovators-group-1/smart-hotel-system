# 🍽️ QuickDine

**QuickDine** is a smart web-based ordering and management system for hotels, restaurants, and bars.  
It enables customers to easily scan a QR code at their table, explore the menu, place orders, and make payments — while administrators and chefs manage orders, track statuses, and analyze performance through an integrated dashboard.

---

## 🚀 Features

### 🧍 For Customers
- Scan unique QR code per table to access the menu.
- Browse categorized meals and drinks.
- Add items to cart and place orders.
- Choose payment method:
  - 💳 Express payment (enter payment number).
  - 💵 Cash payment (auto-generates unique code for receipt).
- Track order preparation and serving status in real time.

### 🧑‍💼 For Admins
- Manage menu items (add, edit, update availability).
- View, confirm, and assign customer orders.
- Generate digital receipts and verify payments.
- Track order history and view reports.
- Access analytics and daily summaries.

### 👨‍🍳 For Chefs
- Receive and view assigned orders from admin.
- Update preparation status (“Preparing”, “Ready”).
- Notify waiters/admin when orders are ready.
- Review completed orders in order history.

---

## 🧱 System Modules

1. **Client Workflow** – Customer menu browsing, ordering, and payment.  
2. **Admin Workflow** – Order management, reporting, and menu configuration.  
3. **Chef Workflow** – Order processing and kitchen coordination.  
4. **System Workflow** – Analytics, logs, and automated report generation.
